Instructions for Building the gopaintCommon package

1.  Install JDK1.11

2.  Install ant version 1.9

3.  Update property files with applicable settings

4.  Execute command 'ant' from goPaintCommon directory.  This will compile the files and build gopaintCommon.jar file in lib_output directory.  If previous builds have been done, use command 'ant clean' to delete files from previous builds.
